## 기능
1. 메인페이지
2. 회원가입
3. 로그인
4. 상품검색
5. 상품상세
6. 장바구니
7. 주문서 작성
8. 결제
9. 주문완료 (취소/배송조회/환불)


## 역할 분담
1. 메인페이지 / 검색기능 / 회원관련기능 / 총괄 / 발표 : 전희민
2. 회원관련기능 / css : 박진아
3. 마이페이지 : 이성한
4. 제품페이지( 제품 리스트 / 제품 상세조회 ) : 최명규
(보류 : 리뷰/상품QnA)
5. 장바구니 / 제품 주문( 주문/배송 정보) / 결제 / 주문내역 : 권유정, 임동국
(보류 : 주문/배송 상태 / 주문 취소/반품)
